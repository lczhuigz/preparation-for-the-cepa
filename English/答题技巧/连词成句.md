# 注意事项

1. 题目一般会给出首字母大写的单词，这往往就是句子的开头，要先锁定好句首位置。

2. 考试中如果题目没有给出标点符号，作答时要自己补全。注意英文句子末尾要写英文句点不要写中文句号"。

3. 如果遇到疑问句，一时不确定语序，可以先把疑问词写出来，再把助动词或系动词提前，剩下的部分按照陈述句的语序去整理。

   

# 做题步骤

如果实在不知道怎么下手，可以按照这个基本步骤来做:

第一步，先找首字母大写的词，确定句子开头;

第二步，找到谓语动词，抓住句子的核心;

第三步，通读剩余部分，理清主语、宾语、状语等成分。

一般按照这三步，句子大致就能顺出来。要是还觉得困难，可以再去看"写句子的十大步骤”
