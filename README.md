# 📚 湖北专升本备考资料库

> 2026 年湖北普通高等学校专升本考试备考资料，目标院校：**湖北文理学院理工学院**（计算机科学与技术方向）。

## 📖 仓库简介

本仓库系统整理了湖北专升本考试的备考资源，涵盖 **大学英语** 和 **C 语言程序设计** 两门核心科目，包含词汇、语法、真题、模拟卷、答题技巧以及 C 语言编程练习等内容，助力高效备考。

## 🗂 内容概览

### 大学英语
- **词汇**：22 个单元核心词汇 + 语境记忆法 + 单词记忆技巧
- **固定搭配**：10 天打卡计划 + 小蓝书汇总
- **语法**：11 张思维导图（词性、句型、时态、从句、倒装等）
- **题型**：阅读理解、完形填空、翻译、写作模板、连词成句
- **真题与模拟**：历年真题 + 好老师/小蓝书/小黑课堂模拟卷

### C 语言程序设计
- **考试大纲**：湖北文理学院理工学院 2026 年官方考纲
- **编程练习**：31 道基础练习 + 猜数字游戏
- **教材参考**：谭浩强《C 程序设计》第五版 + 张玉生版答案
- **模拟试卷**：15 套卷 + 押题卷 + 冲刺卷 + 预测卷

## 🚀 使用建议

1. **先读 [备考指南.md](备考指南.md)**，了解分阶段复习策略
2. **英语**：从单词和语法思维导图入手，结合答题技巧练习真题
3. **C 语言**：对照考纲，边做 `exercise/` 编程题边看教材
4. **冲刺阶段**：集中刷测试试卷，控制答题时间
